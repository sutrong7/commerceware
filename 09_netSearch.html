NetSearch

홈 SOLUTION NetSearch

솔루션 특장점

NetSearch는 온라인 쇼핑몰의 다양한 상품 검색과 추천에 최적화된 검색 솔루션으로 신속하고 정확도 높은 우수한 검색 성능과 다양한 사용자 편의 기능을 제공합니다. 분산 환경에서 유연하게 확장할 수 있어 안정성을 보장하며 편리한 검색 API를 제공하여 다양한 서비스와의 빠른 연동을 지원합니다.

NetSearch 구성도

NetSearch 주요 기능

  • 빠르고 다양한 데이터 수집 및 색인

    File, DB, HTTP 등 다양한 원천 데이터 수집이 가능합니다.

    초기 데이터 색인 후 변경된 데이터만 색인하는 동적 색인을 지원합니다.

    고속의 데이터 색인 후 서비스의 중단없이 실시간으로 검색 결과에 반영됩니다.

  • 각 사이트에 특화된 검색어 사전 관리

    사이트 별 사전을 따로 관리할 수 있어 사이트 성격에 맞는 사전을 제작합니다.

    사전 수정 후 별도의 서버 재기동 없이 즉시 색인이 가능합니다.

  • 동의어/유의어 기능으로 편리한 사용자 검색

    쇼핑몰에서 검색어로 자주 사용되는 데이터를 축적한 고품질의 동의어/유의어 사전을 제공합니다.

    관리자가 NetSearch 어드민에서 간편하게 동의어, 유의어를 추가할 수 있습니다.

    신조어, 약어 등을 동의어로 설정해 검색 성공 확률을 높여 최상의 검색 품질을 유지합니다.

  • 검색서버 클러스터링으로 안정적 서비스 제공

    일부 서버 서비스 불능 시 클러스터링 재구성으로 24시간 서비스가 가능합니다.

    서비스 중 트래픽 증가 시 유연한 서버 증설이 가능합니다.

    각각의 서버에서 오류 발생 시 지정한 담당자에게 즉시 이메일을 발송합니다.

  • 편리한 검색 API와 다국어 검색 기능

    다양한 플랫폼에서 검색 서비스를 신속하게 출시할 수 있도록 HTTP Restful 방식의 검색 API를 제공합니다.

    각 언어별 형태소 분석기를 통해 유니코드 기반의 다국어 검색을 지원합니다.

    NetSearch 어드민도 다국어 기능을 제공하여 다양한 언어로 운영이 가능합니다.
